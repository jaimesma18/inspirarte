# Inspirarte - Taller de Arte 2026

Sitio web del taller de arte Inspirarte para el año 2026.

## Estructura del Proyecto

```
GimeWeb/
├── index.html          # Página principal
├── images/            # Imágenes (logo, galería)
│   ├── logo.png
│   └── proyecto*.jpg
└── videos/            # Videos
    └── Inpirarte2025.mp4
```

## Cómo usar GitHub Pages

1. Crea un repositorio en GitHub
2. Sube todos los archivos a GitHub
3. Ve a Settings > Pages
4. Selecciona la rama "main" o "master"
5. Tu sitio estará disponible en: `tuusuario.github.io/GimeWeb`

## Notas

- Asegúrate de que `index.html` esté en la raíz del repositorio
- Las carpetas `images/` y `videos/` deben estar incluidas
- Los archivos de instrucciones (.txt) son opcionales
